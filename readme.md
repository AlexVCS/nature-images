This repo is a demo of CSS Grid Lanes.

To view this project on your machine as intended you'll need the Safari Technology Preview.

You'll also need to signup for the [Unsplash API][unsplash-url]. Create a .env file by running this:

```bash
touch .env
```

In your .env file add this (delete the placeholder and add your Unsplash access key):

```bash
VITE_CLIENT_ID=YOUR_UNSPLASH_ACCESS_KEY
```
